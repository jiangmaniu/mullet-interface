隐私政策

发布日期：2025年6月1日

生效日期：2025年6月1日

尊敬的用户：

我们始终将您的隐私保护置于重要位置。为帮助您更好地理解我们在您使用服务过程中对个人信息的处理方式，特制定本政策说明如下：

  • 信息收集与使用

    在您使用本应用服务过程中，我们可能会基于服务需要收集、存储并使用您的相关信息。本政策将具体说明信息处理的范围、目的及您的相关权利，请您仔细阅读（特别是加粗条款）。

  • 自启动功能说明

    为实现消息推送功能，确保应用在关闭或后台运行时仍能正常接收广播信息，本应用需启用自启动能力。系统将定期通过广播唤醒应用完成自启动或关联启动，此为实现服务功能的必要措施。

    • 当您点击推送消息时，在您明确授权同意后，应用将跳转至相关内容页面；
    • 若未获得您的同意，则不会触发关联启动行为。
  • 隐私政策确认

    若您选择继续使用服务，即表示您已充分阅读并同意我们按照本政策处理您的相关信息。

一、信息收集与使用

1. 必要功能授权

以下权限需您主动开启，关闭将影响相关服务：

  • 相机/相册：用于身份认证、上传图片
  • 地理位置：反洗钱合规用途
  • 麦克风：语音输入服务
  • 生物识别：仅验证结果，不收集特征数据
  • 通知权限：接收推送消息

2. 后台数据收集

为优化应用性能，后台可能获取ANDROID ID等设备信息，用于分析崩溃、能耗数据。

二、数据安全措施

  • 存储：境内服务器加密存储，符合国家三级等保要求。
  • 传输：SSL加密技术保障通信安全。
  • 期限：仅保留实现目的所需时间，停止运营后将匿名化处理。

三、您的权利

权利类型 操作方式 例外情况
访问/更正 联系客服 需身份验证
删除/注销 账户设置提交申请 法律要求的保存期

四、其他重要条款

  • 未成年人：需监护人同意后使用服务。
  • 政策更新：将通过App或短信通知，继续使用视为接受修订内容。
